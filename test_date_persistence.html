🔧 كيفية الاختبار

⚠️ خطوات الاختبار

1. افتح Developer Tools (F12) وانتقل إلى Console

2. اختر مطعم ونوع اشتراك وتاريخ بداية

3. اضغط على زر "تعديل" لأي وجبة

4. اختر تاريخ جديد صحيح

5. راقب console.log للتأكد من حفظ التاريخ

6. تأكد من أن التاريخ الجديد يظهر في الواجهة

📊 Console Logs المتوقعة

عند حفظ التاريخ:

📅 تم حفظ التاريخ المخصص: { mealKey: "meal_0", newDate: "2024-01-15", allCustomDates: { "meal_0": "2024-01-15" } }

عند استخدام التاريخ:

📅 استخدام التاريخ المخصص: { mealKey: "meal_0", customDate: "2024-01-15", originalDate: "2024-01-14", finalDate: "2024-01-15" }

النتيجة النهائية:

📅 النتيجة النهائية لأيام الوجبات: [ { label: "الوجبة الأولى", date: "2024-01-15", displayDate: "Monday, January 15, 2024", hasCustomDate: true } ]
