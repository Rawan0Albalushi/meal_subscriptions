🚀 المميزات الجديدة
1. تحسين التخطيط: تم تحسين التخطيط العام للصفحة ليتناسب مع الهواتف المحمولة مع إعادة ترتيب العناصر.

2. تحسين الفلاتر: تم تحسين واجهة البحث والفلاتر لتكون أكثر سهولة في الاستخدام على الهواتف.

3. تحسين النماذج: تم تحسين نموذج إضافة/تعديل أنواع الاشتراكات ليكون أكثر سهولة على الهواتف.

4. تحسين الجداول: تم تحسين عرض الجداول مع إضافة التمرير الأفقي وتحسين عرض البيانات.

5. تحسين الوصولية: تم تحسين التفاعل باللمس والوصولية للمستخدمين ذوي الاحتياجات الخاصة.
