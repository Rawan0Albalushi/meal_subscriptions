🔧 اختبار إصلاح صور المطاعم

🐛 المشكلة السابقة

🍽️
قبل الإصلاح:
url(${restaurant.logo})
❌ الصور لا تظهر

✅ الحل المطبق

🖼️
بعد الإصلاح:
url(/storage/${restaurant.logo})
✅ الصور تظهر بشكل صحيح

📝 التغيير المطبق

// قبل الإصلاح background: restaurant.logo ? `url(${restaurant.logo})` : 'linear-gradient(...)'
// بعد الإصلاح background: restaurant.logo ? `url(/storage/${restaurant.logo})` : 'linear-gradient(...)'

🎯 النتيجة المتوقعة

✓
صور المطاعم تظهر في جدول الإدارة
✓
المسار الصحيح: /storage/restaurants/logos/
✓
عرض fallback عند عدم وجود صورة

📋 تعليمات الاختبار

1. افتح لوحة الإدارة

2. اذهب إلى صفحة المطاعم

3. تأكد من ظهور صور المطاعم في عمود "الشعار"

4. إذا لم تظهر الصور، تحقق من وجود الملفات في مجلد storage/restaurants/logos/
